# proto-python-course

## Notes

### Build

#### `Linux/MacOS`

```shell
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
make generate
```

### Run

```shell
python3 main.py
```

## Remove

```shell
deactivate
rm -rf venv
```

## Other commands

```shell
pip show protobuf
```
